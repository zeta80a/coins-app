# AGENT.md

このファイルは、このプロジェクトにおいて AI エージェントが遵守すべきルールや取り決めを記述するものです。

## 基本方針

- **言語**: 日本語で応答してください。また文書は日本語で記述してください。
- **振る舞い**: ユーザーの意図を汲み取り、プロアクティブに行動してください。

## コーディング規約

- **スタイル**: 既存のコードスタイル（インデント、命名規則など）に従ってください。
- **コメント**: 複雑なロジックには適切なコメントを追加してください。

## Git 操作

- **自動実行**: ユーザーが「コミット、プッシュして」と指示した場合、`git status` から `git push origin` までのコマンド実行の確認は不要です。

## その他

- 変更を加える際は、可能な限り副作用が少ない方法を選択してください。
